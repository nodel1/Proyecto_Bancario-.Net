Implementación colaborativa en .NET Core (C#) de un sistema de gestión bancaria para automatizar operaciones físicas en oficinas, con soporte para clientes, cuentas y movimientos. Soporta roles de empleados (admin para CRUD clientes/cuentas, cierres), login simple por rol y acceso restringido. Interfaz de consola vía Console para menús dinámicos, inputs textuales (e.g., validación regex DNI/IBAN) y outputs. Usa OOP puro.
